# Specification

## Summary
**Goal:** Build a modern, visually rich Indian-themed restaurant website for Indian Garden Chicago with a Motoko backend serving menu and restaurant data.

**Planned changes:**
- Create a full-screen hero section with the restaurant name, tagline, and CTA buttons ("View Menu" / "Reserve a Table") using saffron, turmeric gold, deep crimson, and ivory palette with mandala/paisley SVG motifs
- Add a sticky navigation bar with smooth-scroll links to Menu, About, and Contact sections
- Build a Menu section with category tabs (Appetizers, Breads, Vegetarian Entrees, Non-Vegetarian Entrees, Biryanis, Desserts, Drinks) displaying styled cards with dish name, description, and price using gold accent borders
- Build an About section with a two-column layout (decorative image + narrative text) describing Indian Garden Chicago's history and culinary philosophy
- Build a Contact & Info section with address (68 W. Washington, Chicago, IL), phone number, per-day operating hours, and an embedded Google Maps iframe
- Create a footer with restaurant name/logo text, quick nav links, social media placeholder icons, address, and copyright line in deep crimson/dark saffron with ivory text
- Apply site-wide Indian art theme: Playfair Display for headings, clean sans-serif for body, decorative SVG section dividers, subtle texture overlays, fully responsive layout
- Implement a Motoko backend actor with pre-seeded data and query functions for menu items (by category), restaurant info/hours, and featured/specialty dishes
- Frontend fetches and displays data from the backend actor

**User-visible outcome:** Users can browse an elegantly themed Indian restaurant website showcasing the full menu by category, read about the restaurant's story, find contact info and hours with a map, and navigate fluidly across all sections on any device.
